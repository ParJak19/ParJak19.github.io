BODY Contien les donnée visuelle de la page
TITLE Spécifie le titre du document, affiché dans l'onglet du navigateur.

Javascript var Déclare une variable

const Déclare une variable constante dont la valeur ne peut pas être modifiée.
function Définit une fonction, un bloc de code qui exécute une tâche ou calcule une valeur.
if Introduit une instruction conditionnelle pour exécuter du code en fonction d'une condition.
